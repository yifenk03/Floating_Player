
# 漂浮视频播放器

一个轻量、美观的无边框置顶视频播放器，支持多种视频格式，具备拖放文件、播放列表、音频设备选择、自定义视频比例等功能，适配高 DPI 显示，操作便捷。


![界面](GUI.png)

# 功能特性:

✅ 无边框置顶窗口：始终悬浮在其他窗口上方，圆角设计更美观

✅ 多格式支持：兼容 MP4、MKV、AVI、MOV、WEBM、GIF、WMV、FLV、TS、M2TS 等格式

✅ 便捷文件操作：支持拖放文件 / 文件夹、弹窗选择文件，自动加入播放列表

✅ 播放列表管理：可视化播放列表，支持拖拽排序，双击快速切换播放

✅ 灵活的播放控制：播放 / 暂停、上一曲 / 下一曲、进度调节、音量控制（静音 / 加减）

✅ 音频设备选择：支持切换音频输出设备（需 PyQt5 Multimedia 组件支持）

✅ 视频比例调整：支持 16:9、9:16、4:3 及原始比例切换，适配不同场景
✅ 快捷键操作：全程键盘控制，无需鼠标

✅ 自动隐藏控制栏：鼠标离开后自动隐藏控制栏，仅保留视频画面

✅ 窗口自由调整：支持拖动位置、调整大小（边角 / 边缘均可拖拽）

# 环境要求

Python 3.6+
PyQt5（需包含 QtMultimedia 组件）

# 安装依赖

pip install PyQt5

⚠️ 注意：部分系统（如 Linux）可能需要额外安装 Qt 多媒体插件，例如：
Ubuntu/Debian: sudo apt-get install libqt5multimedia5-plugins

#使用方法

## 1. 运行程序

python floating_player.py

## 2. 打开视频文件

按钮方式：点击控制栏「📂」按钮，选择单个 / 多个视频文件
拖放方式：直接将视频文件 / 文件夹拖到播放器窗口
右键菜单：右键点击播放器，选择「添加文件」/「添加文件夹」
快捷键：按下 O 键快速打开文件选择弹窗

## 3. 播放控制

播放 / 暂停：点击「▶/⏸」按钮 或 按 空格 键
上一曲 / 下一曲：点击「⏮/⏭」按钮
进度调节：拖动进度条 或 按「←/→」键（每次 ±10 秒）
音量控制：拖动音量条 / 按「↑/↓」键（每次 ±5%） / 按 M 键静音

## 4. 播放列表

显示 / 隐藏：点击控制栏「📋」按钮
切换播放：双击列表中的视频项
排序：拖拽列表项调整播放顺序

## 5. 窗口操作

拖动位置：点击视频区域（非边角）并拖拽
调整大小：鼠标移至窗口边角 / 边缘，光标变化后拖拽
最小化：点击「─」按钮 或 系统最小化操作
退出：右键菜单选择「退出」 或 按 ESC 键

## 6. 高级设置

视频比例：右键菜单「视频比例」选择对应比例（16:9/9:16/4:3 / 原始）
音频设备：右键菜单「音频输出设备」选择可用设备（无选项则表示不支持）


# 自定义配置
可直接修改代码顶部的核心配置项，无需改动逻辑：

## 窗口圆角半径
WINDOW_ROUND_RADIUS = 4
## 支持的视频格式
SUPPORT_FORMATS = ['.mp4', '.mkv', '.avi', '.mov', '.webm', '.gif', '.wmv', '.flv', '.ts', '.m2ts']
## 控制栏自动隐藏延迟（毫秒）
AUTO_HIDE_DELAY = 1500

# 注意事项

* 音频设备选择功能依赖 QAudioOutputSelectorControl，部分 PyQt5 版本 / 系统可能不支持，属正常现象
* GIF 文件仅支持播放，进度调节可能无效（GIF 格式特性）
* 窗口置顶模式下，部分全屏应用可能覆盖播放器（系统层级限制）
* 高 DPI 屏幕已做适配，若显示异常可调整系统缩放比例
* 点击右下角按钮可最小化，最小化到屏幕左下角
